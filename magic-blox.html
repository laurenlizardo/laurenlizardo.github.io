Overview

Magic Blox is a block-building simulator in AR! Users build their own block structures wherever they please in the game’s Sandbox Mode AND find hidden blocks in Hide ‘n’ Seek Mode AND solve puzzles in Puzzle Mode. So many choices!

Reponsibilities

  • Co-developed the game’s core mechanics and gameplay systems within a one-week prototyping phase
  • Created an instantiation system that formed a different structure of blocks for every game session in the game’s Hide ‘n’ Seek Mode
  • Provided additional work for visual effects and sounds

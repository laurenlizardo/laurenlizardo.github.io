Overview

Blown Away is an AR face-tracking game that replaces the user’s face with that of a pufferfish! By controlling the pufferfish using facial expressions, users can blow away the various sea creatures that swim by. Higher points are rewarded if the same kind of sea creature is targeted twice or three times in a row!

Reponsibilities

  • Iterated on UX design to solve problems surrounding the user’s ability to estimate depth in AR
  • Utilized the in-house face-tracking API built on top of the ARKit and ARCore frameworks to prototype an inhale/exhale game based on the formations of the user’s mouth
  • Manipulated character blendshape values so that gameplay events conveyed proper facial expressions and behaviors
  • Created a sea creature base class that defined and executed general movement capabilities and physics behaviors
  • Modified the derived sea creature classes so that each character was visually and behaviorally distinct from one another
  • Implemented game systems such as a Match 3 scoring system and an item unlocking system to be used by all company projects
  • Integrated Unity notifications and AppsFlyer ad banners
